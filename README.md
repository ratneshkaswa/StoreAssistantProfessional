# Store Assistant Professional

Windows desktop POS for Indian retail clothing stores.

Stack: WPF + BlazorWebView + MudBlazor + (SQLite + EF Core coming when we need storage).

## Run

```
dotnet run --project src/StoreAssistantProfessional
```

## Status

Day 1. Fresh start. The old codebase lives at `../StoreAssistantPro/` and is kept for reference — features port over one at a time, on demand.
